# AI Resume Analyzer

An intelligent web application designed to bridge the gap between job seekers and Applicant Tracking Systems (ATS). This tool provides real-time feedback, scoring, and career coaching to help users optimize their resumes for modern hiring workflows.

## 📌 Overview

This project is a Resume Analyzer web application that helps users:

* Upload resumes
* Get ATS score
* Check job match
* Receive improvement suggestions
* Get career guidance

---

## 🚀 Features

* Resume Upload
* ATS Score Analysis
* Job Matching
* Rewrite Suggestions
* Career Coaching

---

## 🛠 Tech Stack

* Frontend: React (Vite)
* Backend: Node.js (Express)

---

## 📁 Project Structure

> Note: Files are currently placed in a single folder (not separated into frontend/backend)

* `/src` → Frontend (React components)
* `/public` → Static files
* `/index.html` → Main HTML file
* `/vite.config.js` → Frontend configuration
* `/server.js` → Backend API logic
* `/routes/` → API routes (if present)
* `/services/` → Resume analysis logic (if present)
* `/package.json` → Project dependencies

---

## ▶️ How to Run

### 🔹 Run Frontend

npm install
npm run dev

Open in browser:
http://localhost:5173/

---

### 🔹 Run Backend

node server.js

Runs on:
http://localhost:3001/
